Instructions
'''''''''''''''''

1. You will need to install Node.js and Ruby. (If you have this already skip to 2) 

(for windows)
https://nodejs.org/en/download/
http://rubyinstaller.org/downloads/  (during the install tick allow ruby access from PATH

Now Start the command prompt with Ruby and enter 

Locate the newly Ruby command prompt (most probably on your desktop). Click Open the command prompt with Ruby
Type the following into the ruby command line and exit once installed:- 
    gem install sass


2.  Open a normal instance of command prompt and type the following:
    npm install
    bower install
    gulp

A local server will be created at localhost:3000 and will begin load the page. If you have any troubles getting
the project loaded correctly please do not hesitate to give me a call.
